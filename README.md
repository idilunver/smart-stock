# ☕️ SmartStock: Barista Envanter Ekosistemi

SmartStock, butik kahve dükkanları ve barista profesyonelleri için tasarlanmış, modern bir envanter yönetim ve bilgi paylaşım platformudur. Generic kurumsal tasarımlardan uzak, sıcak "Caffeine-Inspired" estetiği ile operasyonel verimliliği birleştirir.

##  Öne Çıkan Özellikler

###  Akıllı Envanter Yönetimi (Admin)
- **Gerçek Zamanlı Takip:** Firebase Firestore entegrasyonu ile stok verileri anlık güncellenir.
- **Dinamik Grafikler:** Chart.js ile görselleştirilmiş stok analizi.
- **Kritik Stok Uyarıları:** 10 adedin altına düşen ürünler için otomatik görsel uyarılar.
- **CSV Dışa Aktar:** Tek tıklamayla tüm envanteri Excel uyumlu formatta raporlama.

###  Market Sistemi
- **Responsive Arayüz:** Tüm cihazlarda kusursuz alışveriş deneyimi.
- **Sepet Yönetimi:** Dinamik sepet sistemi ve stok kontrolü (stokta kalmayan ürünler için otomatik engelleme).
- **Arama & Filtreleme:** Ürünlere hızlı erişim için gelişmiş arama motoru.

###  Barista Bilgi Arşivi (Blog)
- **Profesyonel Rehberler:** V60 demleme, çekirdek DNA'sı ve kavurma teknikleri gibi uzun soluklu rehberler.
- **Premium Okuma Deneyimi:** Glassmorphism detayları ile zenginleştirilmiş, okuma odaklı modal yapısı.

##  Teknoloji Yığını
- **Frontend:** React 19 + TypeScript + Vite
- **Styling:** Tailwind CSS (Custom Coffee Palette)
- **Veritabanı & Auth:** Firebase (Firestore & Authentication)
- **Animasyonlar:** Framer Motion
- **Bildirimler:** React Hot Toast
- **Grafikler:** Chart.js

##  Kurulum

Projeyi yerel bilgisayarınızda çalıştırmak için:

1. Depoyu klonlayın:
   ```bash
   git clone https://github.com/idilunver/smart-stock.git
   ```
2. Bağımlılıkları yükleyin:
   ```bash
   npm install
   ```
3. `.env` dosyası oluşturun ve Firebase API anahtarlarınızı ekleyin.
4. Geliştirme sunucusunu başlatın:
   ```bash
   npm run dev
   ```
